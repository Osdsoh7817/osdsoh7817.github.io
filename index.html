📖 What is Score2ConVec

Score2ConVec is a tool for voice synthesis. It helps you change MIDI files and lyrics into singing voice data. You can use this data with voice conversion models like RVC or so-vits-svc. It bridges the gap between text and musical notes to create human-like singing results.

💻 System Requirements

Your computer needs these things to run the software smoothly:

  • Windows 10 or Windows 11.
  • A graphics card from NVIDIA with at least 6GB of video memory.
  • At least 16GB of system memory.
  • Enough storage space for model files and data.

🚀 How to Download and Install

Follow these steps to set up the software on your Windows machine:

  1. Visit the official releases page to see the latest version.
  2. Look for the section labeled Assets.
  3. Click the link that ends in .zip or .exe to download the installer.
  4. Find the file in your Downloads folder once the transfer finishes.
  5. Double-click the file to open the setup window.
  6. Follow the instructions on your screen to finish the installation.

⚙️ How to Use the Software

Once you install the software, you can start creating. Place your MIDI files in the input folder. Add your lyrics in a text file. Select your preferred voice model from the settings menu. Press the Start button to begin the processing. The software will create the output file in your designated folder.

🧪 Troubleshooting Common Issues

If you see an error message, check these areas first:

  • Ensure you have the latest drivers for your graphics card.
  • Make sure your MIDI files contain melodic notes.
  • Check if your path contains spaces or special characters, as these can cause issues.
  • Verify you have enough space on your hard drive before you start a new task.
